## 失败用例详情

### AT-7.7 LLM 不可用降级

- **现象**: 未执行测试
- **预期**: LLM 不可用时显示降级模板结论("AI 综合研判暂不可用..."), 总分=纯公式得分
- **实际**: 当前环境 LLM 可用, 无法在不修改代码/配置的情况下模拟 LLM 不可用
- **根因**: 测试环境限制, 需要设置 LLM_JUDGE_ENABLED=false 重新运行检测才能验证
- **影响范围**: 降级模板展示逻辑; 不影响核心检测功能
- **建议修复优先级**: P2 (L2 层已有覆盖, 仅缺 UI 级验证)

### AT-9.2 维度级复核

- **现象**: 所有维度的复核请求均返回 404 "维度 XXX 无 OA 记录"
- **预期**: 在维度详情页可以对单个维度进行确认/驳回/备注操作
- **实际**: POST /api/projects/:pid/reports/:v/dimensions/:key/review 对全部 11 个维度均返回 404
- **根因**: 维度级复核 API 依赖 OA(Overall Assessment)记录的存在; 当前检测流程生成的报告中, 维度结果存储在 dimension_results 表而非 OA 记录格式, 导致复核 API 找不到目标记录
- **影响范围**: 维度级精细化复核功能不可用; 报告级复核(AT-9.1)不受影响
- **建议修复优先级**: P1 (影响审计完整性; 需确认是 API 路由问题还是数据模型不匹配)
