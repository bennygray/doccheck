## C2 关键设计决策(供后续 change 参考)

- **改密后旧 token 立即失效 = pwd_v 毫秒时间戳** 方案(不引 Redis 黑名单);详见 `openspec/changes/auth/design.md` D3
- **账户锁定 = `locked_until` TTL 字段 + `SELECT FOR UPDATE`** 原子化,TTL 过后自动解锁(design.md D4)
- **初始管理员 seed = alembic data migration + `ON CONFLICT DO NOTHING`** 幂等(design.md D6)
- **前端路由守卫 = `<ProtectedRoute>` + `<RoleGuard>` 两组件分离**;axios interceptor 等价物用 `fetch` wrapper + 401 回调(design.md D7/D8/D9)
- **E2E globalSetup = `backend/scripts/reset_admin_for_e2e.py`** 每次 run 前复位 admin,保证测试可重复
